# Comparisons

OpenClaiming overlaps with several existing technologies but focuses on a simpler primitive.

## Verifiable Credentials

W3C Verifiable Credentials define a large credential ecosystem including schemas, registries, and revocation frameworks.

OpenClaiming instead focuses on simple signed statements that applications interpret.

## Decentralized Identifiers (DIDs)

DIDs define decentralized identity identifiers and resolution methods.

OpenClaiming can operate independently of DIDs but can also be used to sign DID-related statements.

## DNSSEC

DNSSEC signs DNS records to authenticate domain data.

OpenClaiming operates at the application layer and can represent arbitrary claims beyond DNS data.

## JWT

JSON Web Tokens are widely used authentication tokens. OpenClaiming differs by emphasizing canonical JSON signatures and decentralized publication.
